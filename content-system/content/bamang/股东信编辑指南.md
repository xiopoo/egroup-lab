# 巴菲特股东信编辑指南

## 一、编辑原则

### 1. 核心原则
- **保持原始内容完整性**：不改变核心信息和数据
- **提升可读性**：优化结构和格式，便于阅读和理解
- **保持专业风格**：参考对标站点的设计风格
- **尊重原文语气**：保留巴菲特独特的写作风格和幽默感

### 2. 编辑目标
- 添加清晰的标题和副标题系统
- 进行段落分层处理
- 添加段落间的过渡和连接
- 优化排版和格式
- 确保内容易于阅读和理解

## 二、标题系统规范

### 1. 标题层级
```
# 一级标题：年份股东信主标题
## 二级标题：主要章节标题
### 三级标题：子章节标题
#### 四级标题：细节说明标题
```

### 2. 标题命名原则
- **简洁明了**：用简短的词语概括内容
- **逻辑清晰**：标题之间要有逻辑关系
- **层次分明**：不同层级标题要有明显区分
- **关键词突出**：在标题中突出关键信息

### 3. 标题示例
```
# 2024 巴菲特致股东信
```

## 三、段落分层处理

### 1. 段落长度控制
- **理想长度**：每个段落控制在100-200字
- **长段落处理**：超过300字的段落需要拆分
- **短段落保留**：关键信息可以用短段落突出

### 2. 段落分层原则
- **主题段落**：每个段落围绕一个主题
- **逻辑连接**：段落之间要有逻辑关系
- **过渡语句**：添加适当的过渡语句
- **信息密度**：控制每个段落的信息密度

### 3. 段落分隔符号
使用以下符号进行段落分隔：
```markdown
---

* * *

> **重点提示**
```

## 四、格式优化标准

### 1. 文字格式
- **重点内容**：使用粗体标注 **重点内容**
- **关键数据**：使用斜体标注 *关键数据*
- **引用内容**：使用引用格式 > 引用内容
- **英文内容**：保留原始英文，必要时添加中文注释

### 2. 表格格式
```markdown
| 项目 | 2024年 | 2023年 |
|------|--------|--------|
| 保险承保 | $9,020 | $5,428 |
| 保险投资收入 | $13,670 | $9,567 |
```

### 3. 列表格式
- **有序列表**：用于步骤说明
- **无序列表**：用于要点列举
- **嵌套列表**：用于复杂内容说明

### 4. 引用格式
```markdown
> "点名表扬，分类批评。" —— 汤姆·墨菲

> **重要提示**：本段落强调的关键信息
```

## 五、内容结构要求

### 1. 开篇结构
```markdown
# [年份] 巴菲特致股东信

## 核心要点概览

> **一句话总结**：[年份]股东信的核心内容摘要

---

## 致股东

[开篇致辞段落]
```

### 2. 主体结构
每个主要章节应包含：
- **章节标题**：清晰概括章节内容
- **导读段落**：简要说明章节重点
- **详细内容**：分段详细阐述
- **数据支撑**：用表格或数据说明
- **总结段落**：概括章节要点

### 3. 结尾结构
```markdown
## 年度股东大会安排

[会议安排详情]

---

## 附录：业绩对比表

[业绩数据表格]

---

**撰写日期**：[日期]
**撰写人**：沃伦·E·巴菲特
**职位**：董事长
```

## 六、可读性提升方法

### 1. 添加导读
在每个主要章节开始前添加导读：
```markdown
> **本节要点**：本章节主要讨论[主题]，重点包括[要点1]、[要点2]和[要点3]。
```

### 2. 添加过渡语句
在段落之间添加过渡：
```markdown
接下来，让我们看看...

在了解了...之后，我们再来看看...

值得注意的是...
```

### 3. 添加注释说明
对专业术语添加注释：
```markdown
财产意外险（P/C）是指[简要说明]

浮存金是指保险公司持有的可用于投资的资金
```

### 4. 添加关键信息标注
```markdown
**关键数据**：2024年缴税268亿美元，创美国企业历史纪录

**重要决策**：明确格雷格·阿贝尔即将接任CEO
```

## 十、注意事项

### 1. 编辑禁忌
- **不要删除原始内容**
- **不要改变数据和数字**
- **不要修改引用内容**
- **不要改变原文语气**

### 2. 编辑建议
- **保持原文风格**：巴菲特独特的幽默和比喻
- **突出关键信息**：用格式标注重要内容
- **优化阅读体验**：合理的段落长度和过渡
- **添加必要注释**：对专业术语的简要说明

### 3. 特殊处理
- **英文内容**：保留原始英文，必要时添加中文
- **表格数据**：保持原始格式，优化排版
- **引用内容**：保持原始引用，添加出处标注
- **签名日期**：保持原始格式，放在文档结尾

# 巴芒复利书房资料域

这里是从 `bamangBOOK/content` 导入到 HJ 的巴菲特、芒格与长期投资资料域。

它服务于：

- 巴芒研究与交互式学习；
- 公众号选题和专题文章素材；
- 企业经营、资本配置和复利思想的跨域研究；
- 内容单元、概念关系和引用依据的长期沉淀。

它不服务于：

- 灰金客户案例；
- 灰金企业诊断结论；
- 灰金法律合规意见；
- 未经核验的投资建议或证券推荐。

## 目录

- `articles/`：专题文章
- `letters/`：伯克希尔股东信
- `partnership/`：巴菲特合伙人信
- `talks/`：演讲
- `interviews/`：访谈
- `qa/`：股东大会问答
- `concepts/`：概念
- `companies/`：公司研究
- `people/`：人物
- `books/`：书籍摘要
- `indexes/`：原项目索引副本
- `股东信编辑指南.md`：巴芒资料编辑参考，不是对外发布模板

原项目的 PDF、网页清洗稿、重复内容树和图谱缓存不在这里。需要它们时，先回到原项目核验来源，再做有记录的导入。

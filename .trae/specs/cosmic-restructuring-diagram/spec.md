# 困境重组与诊断星系图 Spec

## Why
首页右侧视觉区域当前展示"企业·业务·财务·债务"辐轮图，信息维度单一，缺少对"困境重组与诊断"核心业务的系统性视觉表达。需要替换为具有天体系统隐喻的多层环形图表，直观展示诊断→重组→重生的完整路径，增强品牌专业深度和视觉冲击力。

## Design Concept: "企业困境重组星系"
仿太阳系结构，以困境重组与诊断为核心，双轨道环承载诊断与重组两大阶段，最外环象征重生/可持续发展。

### 结构层次（从内到外）

```
          [运营] ←─┐
                   │   [资产重组]
  [债务] ←── [困境重组] ──→ [业务重整]
           ── 与诊断 ──
  [资产] ←──┤        ├──→ [债务重组]
                   │
              [重生/价值] ←─ (最外层轨道环)
```

**颜色方案（红色渐变背景 #AB1942→#8B1435 上）**
| 元素 | 颜色 | 说明 |
|------|------|------|
| 核心"困境重组·与诊断" | 金色渐变 #F5B041→#E67E22 + 发光 | 太阳意象 |
| 内轨道环 | rgba(255,255,255,.15) 虚线 | 诊断阶段轨道 |
| 内节点×3（资产·债务·运营） | 青蓝色 #5B9BD5 | 诊断维度 |
| 外轨道环 | rgba(255,255,255,.2) 点线 | 重组阶段轨道 |
| 外节点×3（资产重组·债务重组·业务重整） | 白色半透明 rgba(255,255,255,.9) | 重组方案 |
| 最外环 | rgba(255,255,255,.08) | 重生/价值路径 |
| 游走点 | 白色发光圆点 | 沿轨道运行的"行星" |

### 动效设计
1. 内轨道环：20s 顺时针旋转
2. 外轨道环：25s 逆时针旋转
3. 核心"太阳"：呼吸脉动（发光强度变化）
4. 行星游走点：沿各自轨道环绕运行

## ADDED Requirements

### Requirement: 星系图完整显示
系统 SHALL 在首页 hero 右侧区域完整显示三层轨道结构。

#### Scenario: 桌面端
- **WHEN** 用户打开首页
- **THEN** 右侧显示三层轨道星系图：中心太阳"困境重组与诊断"+ 内轨3节点 + 外轨3节点 + 游走点
- **AND** 各轨道以不同速度和方向旋转

### Requirement: 太阳核心动效
- **WHEN** 桌面端显示
- **THEN** 中心太阳有金色渐变背景 + 发光脉搏动画

### Requirement: 轨道运行
- **WHEN** 桌面端显示
- **THEN** 内轨道（诊断）和外轨道（重组）以不同速度和方向持续旋转
- **AND** 有游走点沿轨道运动

## REMOVED Requirements
- 移除旧辐轮结构（`.radial-center` 的"企业"、`.radial-node-1/2/3` 的"业务·财务·债务"）
- 移除旧连接线（3条 SVG line + 对应的流动动画）

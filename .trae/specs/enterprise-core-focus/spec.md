# 企业核心视觉聚焦优化 Spec

## Why

当前首页 hero 右侧的企业核心动态关系图中，"企业"核心节点占比仅 24%（约 91px），在 380px 容器中视觉占比偏低，未能充分确立企业的视觉主导地位。外层节点与核心之间的距离偏大，导致视觉焦点分散，企业主题关联性和品牌统一性不足。需要聚焦优化——强化企业的视觉中心地位，使所有元素围绕企业形成更紧密、更有层次的展示结构。

## Design Concept: "企业为王"视觉聚焦

保留同心圆双圈结构，但大幅提升核心"企业"的视觉权重，通过增大尺寸、增强发光、添加品牌标识关联、缩小轨道间距等手段，使"企业"成为不可撼动的视觉焦点，所有内外节点清晰呈现"围绕企业"的从属关系。

### 核心优化方向

| 优化项 | 当前值 | 目标值 | 收益 |
|--------|--------|--------|------|
| 核心尺寸 | 24% (≈91px) | 32% (≈122px) | 视觉占比提升33% |
| 核心字号 | 22px | 28px | 更醒目的标题感 |
| 核心边框 | 2px solid | 3px solid + 内发光 | 增强轮廓 |
| 核心发光 | box-shadow 30px | box-shadow 60px | 光晕更明显 |
| 内轨道宽度 | 48% | 44% | 节点更靠近核心 |
| 外轨道宽度 | 76% | 68% | 整体更紧凑 |
| 品牌关联 | 无 | 核心下方加"灰金重组"小字 | 品牌统一性 |

### 颜色方案调整

| 元素 | 当前 | 目标 | 说明 |
|------|------|------|------|
| 核心背景 | rgba(255,255,255,.15) | rgba(255,255,255,.18) | 略提升不透明度 |
| 核心边框 | rgba(255,255,255,.4) | rgba(255,255,255,.6) | 更显眼 |
| 核心发光 | rgba(255,255,255,.15) | rgba(255,255,255,.3) + 品牌色 | 加入品牌红色调 |
| 内轨道环 | rgba(255,255,255,.18) | rgba(255,255,255,.22) | 略提升可见度 |
| 外轨道环 | rgba(255,255,255,.12) | rgba(255,255,255,.16) | 略提升可见度 |
| SVG 连接线 | rgba(255,255,255,.2) | rgba(255,255,255,.3) | 更清晰 |

### 布局参数调整

| 参数 | 当前 | 目标 |
|------|------|------|
| 核心直径 | 24% | 32% |
| 内轨道直径 | 48% | 44% |
| 外轨道直径 | 76% | 68% |
| 内节点尺寸 | 80×36px | 82×38px |
| 外节点尺寸 | 96×40px | 100×42px |
| 核心→内节点距离 | 24%半径 → ~46px | 22%半径 → ~42px |
| 核心→外节点距离 | 38%半径 → ~72px | 34%半径 → ~65px |

### 内节点位置重算（半径 22%）

- 业务（顶部）：`left:calc(50% - 41px);top:calc(50% - 22% - 19px)`
- 财务（120°）：`left:calc(50% + 22%*0.866 - 41px);top:calc(50% + 22%*0.5 - 19px)`
- 债务（240°）：`left:calc(50% - 22%*0.866 - 41px);top:calc(50% + 22%*0.5 - 19px)`

### 外节点位置重算（半径 34%）

- 资产重组（顶部）：`left:calc(50% - 50px);top:calc(50% - 34% - 21px)`
- 债务重组（120°）：`left:calc(50% + 34%*0.866 - 50px);top:calc(50% + 34%*0.5 - 21px)`
- 业务重组（240°）：`left:calc(50% - 34%*0.866 - 50px);top:calc(50% + 34%*0.5 - 21px)`

### SVG 连接线重算（380×380 viewBox）

内节点半径 ≈ 22% × 380 ≈ 84px：
- 顶部：(190, 190-84=106)
- 右下：(190+84*0.866=263, 190+84*0.5=232)
- 左下：(190-84*0.866=117, 190+84*0.5=232)

外节点半径 ≈ 34% × 380 ≈ 129px：
- 顶部：(190, 190-129=61)
- 右下：(190+129*0.866=302, 190+129*0.5=255)
- 左下：(190-129*0.866=78, 190+129*0.5=255)

## What Changes

- 修改 `public/index.html` 中 `.diagram-core` 相关 CSS 属性（尺寸、边框、发光、字号）
- 修改轨道环宽度百分比（44% / 68%）
- 修改轨道环边框透明度（.22 / .16）
- 重新计算并更新 6 个节点的 `left`/`top` 内联样式
- 重新计算并更新 SVG 连接线坐标
- 在核心节点下方添加"灰金重组"小字（品牌关联）
- 可选：核心发光中加入品牌色色调（rgba(171,25,66,.15) 叠加）

## Impact

- Affected code: `public/index.html` CSS 区块（~lines 72-116）和 HTML 区块（~lines 365-420）
- 不改变整体结构，仅调整尺寸、间距、视觉权重
- 交互功能（tooltip / 点击面板）不受影响
- 移动端隐藏不变

## Changed Requirements

### Requirement: 企业核心强化
核心 SHALL 尺寸从 24% 增大至 32%，字号从 22px 增至 28px。

#### Scenario: 桌面端显示
- **WHEN** 用户在桌面端打开首页
- **THEN** 核心"企业"占比约 122px（380px 容器中），成为最醒目的视觉元素
- **AND** 核心发光范围扩大，明显覆盖到内轨道区域
- **AND** 核心下方显示"灰金重组"小字（品牌关联）

### Requirement: 轨道间距紧凑化
- **WHEN** 桌面端显示
- **THEN** 内轨道从 48% 缩至 44%，外轨道从 76% 缩至 68%
- **AND** 节点位置同步调整，整体结构更紧凑
- **AND** 连接线坐标同步更新

### Requirement: 节点尺寸微增
- **WHEN** 桌面端显示
- **THEN** 内节点从 80×36px 增至 82×38px
- **AND** 外节点从 96×40px 增至 100×42px

## Removed Requirements
无移除项，仅做视觉增强调整。
